游览器推荐

游览器推荐

由于电脑沾的比较多，所用的软件也多，原来的不知道用什么游览器现在看来也迎刃而解了。

我便来给大家推荐一些游览器，以防不知道用什么好。

NO.1 微软 Microsoft Edge

简称“ED游览器”。

适用系统：

win7/win10（其他的没试）

由于win7暂停服务，所以不再更新，但仍可以使用。

优点：

  1. 搜索很全，网址栏也可以用bing搜，很方便。

  2. 可以登微软账户（邮箱就行），可以同步用户设置和游览书签等。

  3. 界面简洁，无弹窗广告和游览广告。

  4. 下载有官方，防捆绑。

  5. 自带的翻译很快，很准 ，很狠 。

缺点

在有的机房连接不到微软，因为网络堪忧。

NO.2 Google Chrome

又叫“谷歌游览器”。

适用系统

win7/win10（其他的没试）

正在持续更新。

优点

  1. 一直在持续更新。

  2. 无捆绑，弹窗，广告。

  3. 加载很快。

  4. 兼容绝大部分网站。

缺点

同 ED游览器

NO.3 Firefox

中文叫“火狐游览器”

这是我最推荐在机房使用的游览器

适用系统

win7/win10（其他的没试）

优点

  1. 在机房连不上bing时会用baidu。（除非你非选）

  2. 下载快，最多两分钟，无捆绑。

  3. 无弹窗，无广告。

  4. 使用方便，界面整洁，可以自定义。

  5. 支持众多插件。

缺点

在某些性能方面不如 ED游览器 和 火狐游览器。

NO.4 360游览器

可以说成“360全家桶”

适用系统

win7/win10（你懂的）

优点

  1. 全中文游览器。（包括官方）

  2. 搜索，游览功能强大。

  3. 使用方便。

缺点

会自带“360安全卫士”，“360软件管家”，“360小程序中心”，“360游戏管家”，“鲁大师”等一系列捆绑软件。

NO.5 flash中心

其实它并不是一个游览器，却有游览器的功能

适用 优点

  1. 可以游览所有带flash插件的网页，这是其他游览器做不到的。

  2. 玩游戏方便。

缺点

缺少大部分游览器所需的东西，如插件。

毕竟它并不是一个正规游览器，而是flash下载中心。
